# ケイロミ共和国用語・単語IME辞書

https://user-images.githubusercontent.com/6637993/145686756-86690e64-870b-490f-91f2-65e268da9286.mp4

ケイロミ共和国の関連用語の辞書ファイルです。

次の法則でケイロミワードを変換できるようになります。

- 推しマーク - `：`の後に名前またはあだ名で変換
- タグ - `＃`の後に名前またはあだ名で変換
- ファンネーム - `〜`の後に名前またはあだ名で変換
- ツイッターID - `＠`の後に名前またはあだ名で変換

**具体的な変換例：**

|変換前|変換後|
|---|---|
|：あませ|☂️|
|：あすき|🌹🏰|
|＃りもね|#生りもね|
|＃りもね|#りもあーと|
|～ぺえ|タコ民|
|～ぺえちゃん|タコ民|
|～なあか|おにくたち|
|＠わかば|@touou_wakaba|
|＠なあか|@7naaka|
|あかばみあすき|赤波見遊城|
|かわらべえくま|河童エクマ|
